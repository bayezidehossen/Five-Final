Question-2:What is the difference between innerHTML,innerText,and TextContent?

1.innerHTML:Returns or sets the HTML content inside an element as a string.
2.innerText:Returns or set the visible text inside an element.
3.TextContent:Returns or set the full text content of an element.

Event delegation is a function in javascript where a sigle parent element handels event for its child element.
